# 🥗 Meal-Mate

**헬스케어 식단 기록 및 운동 관리 서비스**

---

## 📌 프로젝트 소개

**Meal-Mate**는 헬스를 즐기는 사용자들을 위한 식단 기록 및 운동 관리 플랫폼입니다.  
개인의 식사 내역과 운동 데이터를 기록하고, 일일 칼로리 현황을 시각화하여 건강한 라이프스타일을 도와줍니다.

## 🛠 주요 기능

### ✅ 사용자 인증
- JWT 기반 로그인 / 회원가입
- Access / Refresh Token 관리

### ✅ 식단 기록 (Meal)
- 식사 이름, 칼로리, 날짜 입력
- 사용자별 식사 내역 조회 API

### ✅ 운동 기록 (Workout)
- 운동 이름, 시간, 칼로리 소모량 기록
- 날짜별 운동 기록 조회
- 평균 칼로리 소모량 계산

### ✅ 통계 기능
- 특정 날짜 기준 일일 총 섭취 / 소모 칼로리 제공
- 날짜 범위 기준 평균 칼로리 분석
